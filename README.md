# Installation
## Flow
Steps listed on the [official Flow installation site](https://flow.org/en/docs/install/)
1. Install compiler
```
npm install --save-dev @babel/core @babel/cli @babel/preset-flow
```
2. Install flow
```
npm install --save-dev flow-bin
```
3. Oh wait can we just use 
```
npm install
```
